## Keep Going

If all the required sections are in place, keep building. Here are directions to explore:

**Add anchor navigation.** If your header has navigation links, add `id` attributes to your sections and use `href="#section-id"` on the links so clicking them scrolls to the right place.

**Build a second card variant.** Create a component that looks different from your main card. A `FeaturedCard` that's larger and more prominent, a `StatBlock` that shows a number and a label, a `QuoteBlock` for a callout, anything that fits your topic.

**Pass more props.** If your card only takes a title and description, add more: an image, a rating, a date, a category label. Richer props make components more interesting to build and style.

**Tighten the styling.** Go through each component and review the CSS carefully. Consistent spacing, readable type sizes, and a coherent color palette make a substantial difference in how finished the site looks. Consider using [CSS keyframes](https://www.w3schools.com/cssref/atrule_keyframes.php) for simple animations.

> **With your partner:** Pick two of these and work through them. When you're done, be ready to show your site to another group and walk them through one component you built: what props it takes, where it's used, and how you styled it.

## Deploy

Get your project live. Follow the React/Vite deploy instructions here: [Hosting with GitHub Pages](/full-stack-web/orientation/05-github-pages/)

Run `npm run deploy` any time you want to push the latest version to your live site.

> **With your partner:** Get both sites live before submitting. Check that the URL loads correctly and that images appear.
