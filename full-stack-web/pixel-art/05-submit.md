## Deploy

Host your editor on GitHub Pages so you can share it and submit it. This is a Vite + React project, so use the **build a static site** method. Run `npm run build` and deploy the `dist` folder.

See [Hosting with GitHub Pages](/full-stack-web/orientation/05-github-pages/) for step-by-step instructions.

## Submit

Submit the link to your live GitHub Pages site on Canvas. Your repository should include a README with a brief description of the editor and the link to the live site. If you completed any challenges, mention which ones in the README.
