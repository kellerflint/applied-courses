Each unit in this course has a Scrimba assignment that goes alongside the in-class work. Scrimba is an interactive learning platform with video lessons and embedded coding challenges. You'll use it to practice React concepts outside of class each week.

## Setting Up

Go to [scrimba.com](https://scrimba.com/) and create a free account using your personal email.

The course is called [Learn React](https://scrimba.com/learn-react-c0e).

Check Canvas for due dates.

## How to Do the Assignments

Scrimba is free for this course. You can watch all the videos without a subscription. At the start of the course, the instructor on Scrimba walks you through how to follow along locally. Do that. Work locally in your own editor using Vite as you go through each section.

When the video pauses for a challenge or practice question, stop and do it yourself before watching the solution. Skipping ahead and watching through defeats the purpose. The challenge is the learning. Scrimba may eventually prompt you to submit challenges for AI feedback, which requires a paid plan. You can skip that prompt and move on to the review after attempting it yourself.

You do not need to do the solo projects. Those require a paid subscription and are not part of this course. Everything else, you should do and follow along with.

## What to Submit

For each Scrimba assignment, submit a link to your deployed GitHub Pages site on Canvas for the section project. The submission should be a live, working URL for the web app, not a link to the GitHub repository.

## Time Estimates

Scrimba shows time estimates for each section. Treat those as optimistic. Plan for the actual time to be 2-3x longer once you include the coding challenges and building the project locally. Start these early! Rushed learning is typically fragile.
