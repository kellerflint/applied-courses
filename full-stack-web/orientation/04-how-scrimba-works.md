Each unit in this course has a Scrimba assignment that goes alongside the in-class work. Scrimba is an interactive learning platform with video lessons and embedded coding challenges. You'll use it to practice React concepts outside of class each week.

## Setting Up

Go to [scrimba.com](https://scrimba.com/) and create a free account using your personal email.

Your instructor will provide the link to the specific course on Canvas. The course is **Learn React**. Make sure you're in the right one (not Learn Next.js or Learn Tailwind CSS).

## How to Do the Assignments

Scrimba has a built-in code editor, but you won't use it. Instead, you'll build the projects locally in Vite and deploy them to GitHub Pages.

**The process for each assignment:**

1. Watch the Scrimba video lessons for the assigned section
2. Complete the embedded challenges inside Scrimba as you go. These are short exercises within the video player and they are required, not optional
3. When you reach a project in the section, build it locally in Vite instead of in Scrimba's editor
4. Deploy the finished project to GitHub Pages
5. Submit the live URL on Canvas

## What to Submit

For each Scrimba assignment, submit a link to your deployed GitHub Pages site on Canvas. The submission should be a live, working URL, not a GitHub repository link.

## Time Estimates

Scrimba shows time estimates for each section. Treat those as optimistic. Plan for the actual time to be 2-3x longer once you include the coding challenges and building the project locally. Start these early in the week, not the night before.
