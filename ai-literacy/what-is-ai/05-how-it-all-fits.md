These four terms nest inside each other. Generative AI is also deep learning. Deep learning is also neural networks. Neural networks are also machine learning. Machine learning is also AI. A single tool can be described by all four labels at once.

{% activity "ai-hierarchy.html", "The AI Landscape", "560px" %}

The note at the bottom of that diagram is the key point. ChatGPT is generative AI, deep learning, a neural network, machine learning, and AI, all at the same time. Which label you reach for depends on which aspect you want to talk about.

This matters because the word "AI" gets used to describe everything from a hand-coded phone tree to a system trained on trillions of words. Knowing the categories lets you ask better questions when someone says "this uses AI": What kind? Did it learn from data? Is it a neural network? Does it generate new content, or does it predict a category?

> **Reflect:** Think of an AI tool you use or have heard about. Work through the hierarchy. Is it rule-based, or does it learn from data? If it learns, is it a neural network? Is it generative? How far down the nesting does it go?

A few examples to check against:

- **GPS navigation:** AI — finds optimal routes — but rule-based and algorithmic. Not machine learning.
- **Netflix recommendations:** Machine learning. Learns from your watch history to predict what you'll like. Not generative.
- **Siri voice recognition:** Neural networks. Trained on audio to recognize words.
- **ChatGPT:** Generative AI, deep learning, neural network, machine learning, and AI. All five labels apply.
- **Spam filter:** Machine learning. Trained on labeled examples of spam and not-spam. Not a neural network (usually).

</details>
