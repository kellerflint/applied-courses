**Machine Learning** is a subfield of AI where the machine learns patterns from data. A person provides examples, and the machine figures out the pattern on its own. Predicting house prices from past sales, flagging spam emails, and recommending videos are all machine learning applications.

The machine learning approach differs from rule-based AI in one key way. A person did not write the rules. The machine discovered them from data.
